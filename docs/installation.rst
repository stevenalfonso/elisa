Installation
============

Requirements
------------

- Python >= 3.9

From Source
-----------

.. code-block:: bash

   git clone https://github.com/stevenalfonso/elisa.git
   cd elisa
   pip install -e .

Optional Dependencies
---------------------

Corner plots and Jupyter support:

.. code-block:: bash

   pip install -e ".[all]"

Development tools (pytest, black, ruff):

.. code-block:: bash

   pip install -e ".[dev]"

Documentation tools (Sphinx):

.. code-block:: bash

   pip install -e ".[docs]"
